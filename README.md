# SmartPDF ToolBox

SmartPDF ToolBox is a Python-based desktop application built using Tkinter.
It provides multiple PDF utilities through a graphical user interface.

## Features
- PDF to DOCX
- Merge PDFs
- Image to PDF
- Compress PDF
- Add page numbers
- Delete pages
- Text to PDF
- Encrypt and Decrypt PDF
- Extract images
- PDF to Audio
- Split PDF

## How to Run
pip install -r requirements.txt
python main.py
